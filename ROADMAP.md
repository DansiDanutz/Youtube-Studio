# Roadmap

Use `pnpm run roadmap:status` to see which items are actually complete, blocked, or still unstarted based on the current code and recorded run artifacts.

## Phase 1: Strong MVP Lane

- Finalize the core run model
- Initialize the implementation repo properly
- Ship brief review and script review
- Ship one deterministic end-to-end Shorts lane
- Validate provider-backed script generation

## Phase 2: Media Generation

- Add narration generation
- Add caption alignment
- Add shot planning
- Add asset generation
- Add render and packaging

## Phase 3: Cockpit Power Tools

- Regenerate one stage without restarting the whole run
- Compare variants side by side
- Swap voice or visual options
- Improve in-product failure explanations

## Phase 4: Reuse and Intelligence

- Searchable run history
- Presets and reusable templates
- Reusable asset and hook library
- Analytics-informed iteration

## Phase 5: Team Workflow

- Better review handoffs
- Role-based workflow improvements
- Multi-run planning and batching
- Publish-readiness and packaging flows

## Product Direction

The long-term goal is not just “AI video generation.”

The goal is a creator operating system that is:

- fast
- observable
- reusable
- human-reviewable
- powerful without becoming hard to use
