# Code of Conduct

## Our Commitment

We want YouTube Studio to be a respectful, ambitious, and constructive open-source project.

Participation should feel:

- welcoming
- serious
- collaborative
- safe for asking questions
- focused on building something excellent

## Expected Behavior

- Be respectful and direct.
- Critique ideas, not people.
- Assume good intent, but do not hide real technical concerns.
- Share context when disagreeing.
- Help make the project easier for others to understand and contribute to.

## Unacceptable Behavior

- Harassment or personal attacks
- Demeaning or exclusionary language
- Bad-faith derailment
- Repeatedly ignoring project scope and contribution norms
- Publishing private or sensitive information without permission

## Enforcement

Project maintainers may remove, edit, or reject contributions and interactions that violate this code of conduct.

## Scope

This applies to:

- issues
- pull requests
- discussions
- review comments
- any project communication spaces

## Reporting

If you encounter unacceptable behavior, open a private report with the maintainers instead of escalating publicly.
